# Books & Thoughts

和 Claude 一起读书，记录思考与对话。

## 使用方式

1. 上传一本书给 Claude
2. Claude 逐章提供摘要和核心思想
3. 基于 Claude 的总结提出问题，展开对话
4. 将思考和笔记记录在对应书籍的文件夹中

## 笔记模板

每本书在对应分类下新建一个文件夹，参考 [_template](./_template/README.md)。
