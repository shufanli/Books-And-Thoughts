# Books & Thoughts

和 Claude 一起读书，记录思考与对话。

## 使用方式

**第一次（只做一次）：**
```bash
bash tools/install.sh
```

**之后每次读书：**
1. 把书（PDF / EPUB / MOBI / AZW3）拖入这个文件夹
2. 自动转换，出现在 `_inbox/`
3. 打开 Claude，说「读 _inbox 里的书」

就这三步，没有其他操作。
